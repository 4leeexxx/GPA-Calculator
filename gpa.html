How Is GPA Calculated?

GPA = Total Quality Points ÷ Total Credit Hours. Each letter grade has a point value (A = 4.0, B = 3.0, etc.). Multiply each grade's points by the course's credit hours to get quality points, sum them up, then divide by total credits.

Standard 4.0 GPA Scale

  • A / A+ = 4.0  ·  A− = 3.7
  • B+ = 3.3  ·  B = 3.0  ·  B− = 2.7
  • C+ = 2.3  ·  C = 2.0  ·  C− = 1.7
  • D+ = 1.3  ·  D = 1.0  ·  F = 0.0
